## üìã Features
This project meets all core requirements and bonus objectives outlined in the task:

* **Create Contacts:** Add new contacts with Name, Email, Phone, and Message.
* **Read Contacts:** View all contacts in a responsive grid layout.
* **Delete Contacts:** Remove unwanted contacts (Bonus Feature).
* **Real-time Validation:** Client-side validation for empty fields and email formats.
* **Numeric Checks:** Enforces numeric input for phone numbers.
* **Responsive Design:** Fully optimized for mobile and desktop screens.
* **State Management:** Managed efficiently using React `useState` and `useEffect`.
